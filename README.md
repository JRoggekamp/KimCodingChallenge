# KimCodingChallenge
Als Productowner wil ik een tool hebben waarmee ik de vestigingen die het meeste in mijn buurt zijn kan zijn.
De applicatie moet een overzicht tonen van maximaal vijf vestigingen bij de gebruiker in de buurt.
De weergave van de vestiging moet minimaal een naam en een adres bevatten.
De gebruiker van de applicatie moet in staat zijn om een vestiging aan te klikken en op die wijze naar de website van de vestiging te navigeren.


Voor deze opdracht mag je gebruik maken van de cms-services acceptatie omgeving. De documentatie kun je vinden op de volgende link.
https://acceptatie.monuta.nl/cms-services/swagger-ui.html
Tevens mag je tijdens deze opdracht geen gebruik maken van frameworks zoals Angular, react, Vue of JQuery.
Het project moet gebouwd worden d.m.v. van HTML 5, CSS 3 en vanilla JavaScript
